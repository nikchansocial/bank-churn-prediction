## Overview

This project focuses on predicting customer churn in the banking sector using machine learning and predictive analytics techniques.

The application analyzes customer demographics, account behavior, and financial indicators to identify patterns associated with customer attrition and support proactive customer retention strategies.

An interactive Streamlit dashboard was developed to transform raw banking data into business intelligence insights and executive-level retention recommendations.

## Business Problem

Customer churn is a major challenge for banks because losing existing customers directly impacts profitability, customer lifetime value, and long-term growth.

Banks need predictive systems capable of identifying high-risk customers early so targeted retention strategies can be implemented effectively.

This project applies machine learning, customer analytics, and business intelligence techniques to predict churn behavior and uncover actionable retention insights.

## Business Insights

* Germany showed the highest customer churn risk among analyzed regions
* Customers aged 46–60 demonstrated elevated attrition behavior
* High-value inactive customers represented major balance exposure
* Customers holding multiple products exhibited higher churn tendencies
* Customer inactivity strongly correlated with retention risk

* Launch targeted retention campaigns in high-risk regions
* Improve engagement strategies for inactive customers
* Develop personalized financial products for vulnerable age groups

## Installation

Clone the repository:

```bash
git clone https://github.com/nikchansocial/bank-customer-churn-prediction-ml.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app/app.py
```
